# 游卡Steam报告 - GitHub Pages 部署指南

## 方案A：使用GitHub Pages（推荐）

### 步骤1：创建GitHub仓库
1. 登录 GitHub
2. 点击右上角 "+" → "New repository"
3. 仓库名：`yoka-steam-reports`
4. 设为 Public
5. 点击 "Create repository"

### 步骤2：上传文件
在新建的仓库页面：
1. 点击 "uploading an existing file"
2. 拖入以下4个文件：
   - index.html
   - Steam新游监控月报0220-电脑版.html
   - Steam新游监控月报0220-手机版.html
   - README.md
3. 点击 "Commit changes"

### 步骤3：开启GitHub Pages
1. 进入仓库的 Settings
2. 左侧菜单找到 "Pages"
3. Source 选择 "Deploy from a branch"
4. Branch 选择 "main" + "/ (root)"
5. 点击 "Save"

### 步骤4：访问网站
等待1-2分钟后，访问：
```
https://你的用户名.github.io/yoka-steam-reports/
```

## 方案B：使用Netlify（更简单）

### 一键部署
1. 访问 https://app.netlify.com/drop
2. 直接拖入整个 `yoka-steam-reports` 文件夹
3. 立即获得一个网址，例如：`https://随机名称.netlify.app`

---

## 方案C：使用Vercel

### 步骤
1. 访问 https://vercel.com
2. 注册/登录
3. 点击 "Add New Project"
4. 选择 "Import Third-Party Git Repository"
5. 输入你的GitHub仓库地址
6. 点击 "Deploy"

---

## 方案D：本地直接打开（如果线上不行）

**重要提示：如果依然打不开，问题是浏览器安全限制。**

### 解决方法1：使用本地服务器
```bash
# 在文件夹里打开命令行，运行：
python -m http.server 8000

# 然后浏览器访问：
http://localhost:8000
```

### 解决方法2：使用VSCode
1. 安装VSCode
2. 安装 "Live Server" 插件
3. 右键 index.html → "Open with Live Server"

## 常见问题

**Q: 点击按钮显示"未找到文件"？**
A: 确保三个文件在同一文件夹，文件名完全一致（包括中文）

**Q: 电脑版打开是空白？**
A: 浏览器按 F12 打开控制台，看是否有错误信息

**Q: GitHub Pages部署后404？**
A: 等待5-10分钟，GitHub需要时间构建
